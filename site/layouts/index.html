What ships

A desktop image with the boring parts already handled.

01

Atomic KDE base

Kinoite-style immutable desktop, rebranded as bluecat and delivered as a bootc OCI image.

02

NVIDIA ready

RPM Fusion NVIDIA stack is built into the image and signed with the bluecat MOK for Secure Boot workflows.

03

Gaming defaults

Steam, Lutris, MangoHud, Gamescope, vkBasalt and Winetricks are included for a practical gaming setup.

04

Developer tools

Distrobox, Nix, virtualization packages, diagnostics and common desktop utilities are ready after install.

05

Offline installer

The rolling ISO embeds the local OCI payload, so the initial deployment does not need a registry pull.

06

Explicit opt-ins

Xbox Wireless Adapter firmware is never bundled; bluecat only downloads it after local user confirmation.

Install

Fresh ISO install or signed rebase.

Use the ISO for a clean install, or rebase an existing Fedora Atomic / Kinoite-style system to the signed bluecat stream. 

curl -fsSL https://raw.githubusercontent.com/echocat/bluecat/main/scripts/rebase-signed | sudo bash
sudo systemctl reboot

# bootc-native systems
sudo bootc switch ghcr.io/echocat/bluecat:44
