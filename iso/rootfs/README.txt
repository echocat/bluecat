bluecat - offline install medium
================================

This is an UNOFFICIAL, independent installer medium for the "bluecat" image
(a Fedora Remix). It is NOT provided by, affiliated with, or endorsed by the
Fedora Project or Red Hat.

The medium boots a Fedora-based Anaconda installer which installs the embedded
bluecat bootc image payload from this ISO. Network access is not required for
the initial deployment.
