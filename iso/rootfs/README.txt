bluecat - network install medium
=================================

This is an UNOFFICIAL, independent installer medium for the "bluecat" image
(a Fedora Remix). It is NOT provided by, affiliated with, or endorsed by the
Fedora Project or Red Hat.

The medium boots a Fedora-based Anaconda installer which then installs the
bluecat bootc image from the container registry over the network.

Project / source : https://github.com/echocat/bluecat
